# Ryzech's Personal Site

This is a Personal blog site, currently a WIP but feel free to look through it :)

## Building & Deploying

To test the website locally:

```bash
zig build serve
```

To build for production:

```bash
zig build website
```

Deployment happens automatically when pushed to sourcehut, through the sourcehut pages option they provide.
